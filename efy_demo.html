Documentation

The logic behind it
  • Minimal code
  • The user is in control, everything is modular
  • the designer combines patterns & will choose defaults
  • It's accessible, touchscreen friendly, convergent
  • Transparency friendly: no popups or multilayered elements when possible
  • Things are easily searchable like closed <details> that open on demand
Variables & Classes
  • --efy_color & --efy_color2 - these create the main gradient of the design
  • Colors use rgb in the 0, 0, 0 format so that they can be reused by other variables & reduce the amount of code
  • Attributes: efy_help , efy_tabs , efy_tab, efy_content
Can I use it on my own projects?

YES! For personal or FOSS projects it's free, for commercial projects you gotta pay $150 once per project. That's it, no monthly fees or limiting your features in exchange for money. You can also donate if you wanna support the project, which will make it cheaper for future users and developers.

You can find out more about the license here: EFY Free & Open Non-Commercial License
